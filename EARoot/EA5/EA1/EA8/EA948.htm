The datatypes described here are conform the XSD data types. <br/>These types are only used, when the characteristics of attributes require a more specific specification. When this is conform specifications of xsd data types they are used.<br/>Examples of these datatypes are anyURI and token.  <br/><br/>When transforming the reference model, no XSD_Types implementation model is generated. <br/>For the XML model, the datatypes are transformed to xsd datatypes.<br/>For implementation (Java) models, it is up to the implementer wether he want to use equivalent java classes which are able to check on the data type.<br/>
