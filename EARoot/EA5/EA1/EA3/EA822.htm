<b>Description.</b><br/>A <b>SwathPattern</b> is a pattern of <b>Swath</b> 's that are worked by an Implement covering a part of or the whole <b>PartField</b>. A SwathPattern is based on an <b>ImplementReferenceLine</b>. A SwathPattern is valid for a specific <b><i>WorkingWidth</i></b>.<br/><br/><b>Remark.</b><br/>The <b>ImplementReferenceLine</b>'s are identical to<i> </i><b><i>FieldSide</i></b> 's).<br/>When only parts of the <i>WorkingWidth</i> are used, <b>SwathSection</b>'s can be specified.<br/><br/><b>DISCUSSION.</b><br/>IS AN IMPLEMENT REFERENCE LINE ALWAYS IDENTICAL WITH A FIELD SIDE? HOW ABOUT A SPRAYING FREE ZONE?<br/><br/>IS IT POSSIBLE THAT SWATHS ARE BASED ON GUIDANCE REFERENCE LINES? IN THAT CASE NOT ONLY WORKING WIDTH BUT ALSO OFFSET TO THE RIGHT OR TO THE LEFT MUST BE SPECIFIED.<br/>
