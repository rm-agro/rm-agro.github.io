<b>Definition.</b><br/>The <b>CropField</b> that preceded the actual CropField.<br/><br/><b>Remark.</b><br/>This object can be used in those cases where no geographic information is available. An actual CropField can be preceded by more CropField's and a CropField can also be followed by more CropField's.<br/><br/>This object can also be used to identify succeeding CropField's. A CropField with the stage "planned" can be preceded by the actual CropField.<br/><br/>The attribute <b><i>Area</i></b> is used to indicate the area of the preceding CropField that is within the borders of the actual CropField,.<br/>
