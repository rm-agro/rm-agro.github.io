<b>Definition.</b><br/>A <b>RectifiedGrid</b> is a Grid for which there is an affined transformation between the grid coordinates and the coordinates of an external coordinate reference system. It is defined by specifying the position (in some geometric space) of the grid "origin" and of the vectors that specify the post locations.<br/><br/><b>Remark.</b><br/>RectifiedGrid is a special case of Grid, which on its turn is defined as:<br/>A Grid implicitly defines an un rectified grid, which is a network composed of two or more sets of curves in which the members of each set intersect the members of the other sets in an algorithmic way. (GML)<br/><br/><br/>
