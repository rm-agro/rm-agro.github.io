<b>Definition.</b><br/>A time period during which a particular <b>Operation</b> is planned to be, or was, executed following a certain specification. <br/><br/><b>Remark.</b><br/>One Operation can be executed in several time periods. It is possible to assign an urgency or timeliness value to each period.<br/><br/>Apart from AbsoluteTiming it is possible to specify a <b>RelativeTiming</b>. One can specify for example that a certain fertilizer gift has to be done three to two weeks before planting.<br/><br/>Apart from AbsoluteTiming it is possible for Task's and Job's to specify one or more <b>TimeFrame</b>'s. AbsoluteTiming has a more agronomical meaning, while TimeFrame has a more (machinery) management meaning.<br/>
