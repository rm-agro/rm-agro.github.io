<b>Definition.</b><br/>A <b>Zone</b> defines a surface in a <b>CropField</b>, <b>ActivityField or Greenhouse</b>. A Zone is described by one or more Polygons and/or a collection of GridPoints.<br/><b><br/></b><b>Remark.</b><br/>A zone is an abstract object which has <b>TreatmentZone</b>, <b>PropertyZone</b>, <b>EnvironmentalProtectionZone, HarvestingZone</b> and <b>DesignatedZone</b> as specializations. <br/>
