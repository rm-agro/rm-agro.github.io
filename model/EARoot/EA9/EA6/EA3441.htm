-- Name --<br/>Sampled Object_ Silage.<br/><br/>-- Definition --<br/>Fermented, high-moisture feed, which is sampled.<br/><br/>-- Description --<br/>Silage is fermented, high-moisture fodder that can be fed to ruminants  or used as a biofuel feedstock for anaerobic digesters. It is fermented and stored in a process called ensiling or silaging, and is usually made from grass crops, including corn (maize), sorghum or other cereals, using the entire green plant (not just the grain). Silage can be made from many field crops.<br/>
