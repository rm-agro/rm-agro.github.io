<br/>        The 'show' attribute is used to communicate the desired presentation <br/>        of the ending resource on traversal from the starting resource; it's <br/>        value should be treated as follows: <br/>        new - load ending resource in a new window, frame, pane, or other <br/>              presentation context<br/>        replace - load the resource in the same window, frame, pane, or <br/>                  other presentation context<br/>        embed - load ending resource in place of the presentation of the <br/>                starting resource<br/>        other - behavior is unconstrained; examine other markup in the <br/>                link for hints <br/>        none - behavior is unconstrained <br/>      <br/>
