A non-abstract bag that is specialized for use as a dictionary which contains a set of definitions. These definitions are referenced from other places, <br/> in the same and different XML documents. In this restricted type, the inherited optional "description" element can be used for a description of this dictionary. <br/> The inherited optional "name" element can be used for the name(s) of this dictionary. <br/> The inherited "metaDataProperty" elements can be used to reference or contain more information about this dictionary. <br/> The inherited required gml:id attribute allows the dictionary to be referenced using this handle. <br/>
