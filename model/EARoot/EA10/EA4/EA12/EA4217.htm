If a feature has a property which takes an array of geometry elements as its value, this is called a geometry array property. A generic type for such a geometry property is GeometryArrayPropertyType. <br/>The elements are always contained inline in the array property, referencing geometry elements or arrays of geometry elements via XLinks is not supported.<br/>
