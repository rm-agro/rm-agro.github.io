<b>Description.</b><br/>A <b>DataSetSeries</b> describes a collection of <b>DataSet</b> 's which logically belong together. This can be two or more different <b>PropertyVariable</b> 's, collected with one sensing system.<br/><br/><br/><b>Example.</b><br/>Reflection in red and reflection in near infra red can be collected by one sensor system, and results in two <b>DataSet</b>s that belong to the same <b>DataSetSeries</b>.<br/><br/>Also remote sensing data over several dates form the same objet (<b>PartField</b>) <br/>
