<b>Definition.</b><br/>A coding list as specified by a standardisation authority.<br/><br/><b>Remark.</b><br/>The (Code)ListIdentifier attribute is of the type IdentifierType, which on its turn has a SchemeIdentifier which can indicate an identiferList which gives a listing of code lists.<br/><br/>This means that the ListIdentifier must be resolvable by its namespace (SchemeDataURI) <br/><br/><br/>
