<b>Description.</b><br/>The <b>PropertyVariable</b> defines a variable which can be measured by a sensor (<b>Operation</b>) , is the result of a (laboratory) analyses or the result of a calculation (<b>DataProcess</b>) . It can be used to describe properties of an Object. Objects can be <b>Field</b>, <b>PartField</b>, <b>PropertyZone</b>, <b>VerticalLayer</b>, (<b>Batch</b> of)<b>Product</b>,  (Batch of)<b>Produce</b>, etc.<br/><br/><b>Remark.</b><br/>Lists of PropertyVariables are maintained by a standardization organization. AgroConnect is such an organization and maintains lists for several <b>PropertyCategory</b>'s. For Some Propertycategory's use is made of standards maintained by other standardization organizations like UN/CEFACT. These organizations are referenced in the PropertyCategory.<br/>
