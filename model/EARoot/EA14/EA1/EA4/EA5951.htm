<b>Definition.</b><br/>A <b>TreatmentZone</b> defines a surface in a <b>CropField</b> or <b>ActivityField</b>, which gets a uniform treatment in that surface by an <b>Operation</b>. A TreatmentZone is described by a MultiSurface.<br/><b><br/></b><b>Remark.</b><br/>By defining more <b>TreatmentZone</b>'s in a <b>CropField</b> or <b>ActivityField</b> it is possible to specify site specific application of <b>Product</b>'s or machine settings.<br/>TreatmentZone can also be used for the Operation "Soil Sampling" to specify the surfaces for which one <b>SoilSample</b> should be collected.<br/><br/>TreatmentZone uses one or more <b>Polygon</b>'s, i.e. a <b>MultiSurface</b>, to describe the surface(s) with uniform treatment.<br/><br/>A TreatmentZone is only intended to specify treatments; i.e. application of products or the specification of working (for example working depth). <br/><br/>To characterize properties of an area, like for example soil type, the <b>PropertyZone</b> is used.<br/><br/>For the collection of Produce, harvesting, the <b>HarvestingZone</b> is used.<br/>
