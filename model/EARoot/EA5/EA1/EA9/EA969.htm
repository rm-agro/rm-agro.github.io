<b>Definition.</b><br/>An identifier is a character string used to uniquely identify one instance of an object within an identification scheme that is managed by an agency. (after UNCEFACT)<br/><br/><b>Remark.</b><br/>The character string is only unique within the scheme. The consequence is that the scheme must be specified with the value (the token) itself. <br/>
