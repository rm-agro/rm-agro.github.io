<b>Definition.</b><br/>The remaining amount of nutrients based on the input quota, minus the already applied nutrients and minus the advised amount of minerals when they would be used.<br/><br/><b>REMARK.</b><br/>This definition requires that when this object is used in a fertilizer recommendation, the farmer must specify the already applied amount of nutrients for all of his CropFields.<br/><br/>This is a redundant class, as it can be calculated from other aqvailable information, and the first part of this remark gives a reason why one should consider not to use this class in a message and leave it to the farm Management System to calculate it.<br/>
