<b>Definition.</b><br/>A <b>PropertyZone</b> defines a surface  which has a uniform PropertyValue, or a range of PropertyValues defined by a minimum and a maximum value for that property zone.  A PropertyZone is described by a MultiSurface. propertyZone is not restricted to the borders of a <b>Field</b>, <b>CropField</b> or <b>ActivityField.</b><br/><b><br/></b><b>Remark.</b><br/><b>PropertyZone</b> uses one or more <b>Polygon</b>'s to describe the MultiSurface with uniform properties.<br/><br/>An example of a <b>PropertyZone</b> is a map of soil types which PropertyZones can extend the borders of a farm.<br/>
