<b>Description.</b><br/>A <b>TrackPattern </b>is used as a reference for a number of tracks to a <b>GuidanceRefferenceLine</b> that should be followed during fieldwork. <br/><br/><b>Remark.</b><br/>The <b>Track</b>'s are parallel to the <b>GuidanceReferenceLine</b> with a certain starting distance ( Offset ) for the first track, and that Offset plus a multiple of the working width for the following Track's. <br/>There are five types of<b> GuidanceReferenceLines</b> conform the <b>GuidancePatternType</b>'s defined in ISO11783. These are AB line, A+ line Curve, Pivot and Spiral.<br/><br/>There can be multiple <b>TrackPattern</b> 's and <b>GuidanceRefferenceLine</b> 's in an <b>ActivityField</b>, as those lines can also be used for the working direction on the headlands.<br/><br/>The <b>GuidanceReferenceLine</b> 's can be used for different <b>TrackPattern</b> 's, as the same line is used for implements with a different working width.<br/>
