<b>Description.</b><br/>A<b> DataProcess</b> describes the processing of an <b>Aggregation</b> of one or more <b>DataSet</b>'s into a new <b>DataSet</b>. Data processing is done following a particular <b>Algorithm</b> and the use of one or more<b> ParameterValue</b>'s which originate from a <b>ParameterFit</b>.<br/><br/><b>Example.</b><br/>A <b>DataSet </b>with the soil pH and another <b>DataSet</b> with nematode population are aggregated and used as input for a nemathode prevention<b> Algorithm</b> and results in a new <b>DataSet</b> with a nematicide application level.<br/>
